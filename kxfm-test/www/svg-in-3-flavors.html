external svg data with integrated CSS instruction and JS tag

can access graph from top window's Javascript context i.e. by:

document.querySelector( "body > object.ktslocal" ).contentWindow.click( "CEP-25" )

document.querySelector( "body > object.ktslocal" ).contentWindow.press( "e" )

external svg data with NO integrated CSS instruction or JS tag

can access graph from top window's Javascript context i.e. by:

document.querySelector( "body >: object.ktslocal" ).contentWindow.onclick( document.querySelector( "body > object.ktsnavig" ).contentDocument, "CEP-25", null)

top window's CSS does not apply here so the result of traversal is not easily visible but there is output in the JS console
